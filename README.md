# Grade Tracker

## About

This program helps students track their grades and study time.

## What it does

You can add subjects, put in your grades, and see how much time you study. It makes charts to show your progress.

## How to run

First install matplotlib:
```
pip install matplotlib
```

Then run:
```
python student_dashboard.py
```

## Menu options

1. Set name
2. Add subject  
3. Add grade
4. Add hours
5. See grade chart
6. See hours chart
7. Save data
8. Exit

## Libraries

- matplotlib (for charts)
- os (for files)
- datetime (for dates) 
- random (for colors)

## Files

- student_dashboard.py (main file)
- data.txt (saves your data)

## How to test

1. Run the program
2. Put your name
3. Add a subject like "Math"
4. Add some grades like 85, 90
5. Add study hours like 3
6. Look at the charts
7. Save when done

You need at least 2 grades to see the grade chart.
